### 2. **PyVISA Oscilloscope Examples**
- **PyVISA Documentation** - [Official Docs](https://pyvisa.readthedocs.io/)
  - Standard library for instrument control across different interfaces (USB, GPIB, Ethernet)
  - Supports ASCII and binary waveform data transfer methods: `read_ascii_values()`, `query_binary_values()`, etc.
- **Practical Example**: Tektronix TBS1072B-EDU over USB showing trigger setup, measurement acquisition, and waveform reading

### 8. **Real-Time Waveform Display Libraries**
**Performance Comparison:**
- **PyQtGraph** (HIGHLY RECOMMENDED)
  - 1000+ FPS update rate (vs 40 FPS for matplotlib)
  - Built on Qt's QGraphicsScene for superior drawing performance
  - PlotWidget class for updating data via `.setData()` method
  - Ideal for oscilloscope applications
  - [PyQtGraph Documentation](https://www.pyqtgraph.org/)
- **pglive** - [PyPI](https://pypi.org/project/pglive/)
  - Thread-safe live plotting wrapper around pyqtgraph
  - Supports PyQt5, PyQt6, PySide6
  - Simplifies real-time plotting implementation
- **matplotlib** - NOT recommended for real-time
  - Too slow for oscilloscope waveforms (40 FPS max)
  - Better suited for static plots
- **VisPy** - [GitHub](https://github.com/vispy/vispy) / [Examples](https://github.com/vispy/vispy/blob/main/examples/demo/scene/oscilloscope.py)
  - GPU-accelerated visualization library
  - Includes oscilloscope demo example
  - Good for very high-speed data visualization

**Standard SCPI Calibration:**
- Use `*CAL?` command to run self-calibration and return results
- Standard SCPI command available on most oscilloscopes
- Implementation depends on U2702A's specific SCPI command set

### 10. **RMS Calculation from Waveform Data**
**Standard Approach:**
```
RMS = sqrt(mean(waveform^2))
```
**Python Implementation:**
- Use NumPy: `np.sqrt(np.mean(waveform**2))`
- Libraries: numpy, scipy.signal, librosa
- RMS provides effective/average value of AC signals
**Additional Signal Processing:**
- **FFT Analysis** - Use scipy.fft or numpy.fft for frequency domain analysis
  - `amplitude_spectrum = np.abs(FFT(signal))`
  - `power_spectrum = np.abs(FFT(signal))**2`
**Resources:**
- [Real Python: Fourier Transforms](https://realpython.com/python-scipy-fft/)
- [NumPy FFT Documentation](https://numpy.org/doc/stable/reference/routines.fft.html)
- [SciPy Signal Processing](https://docs.scipy.org/doc/scipy/reference/signal.html)
